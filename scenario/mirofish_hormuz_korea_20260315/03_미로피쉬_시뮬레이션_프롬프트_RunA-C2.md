# 3. MiroFish 자연어 시뮬레이션 프롬프트 (Run A ~ C-2)
**기본 전제:** 아래 프롬프트는 업로드한 seed 문서(01 + 02-1 ~ 02-4)와, 각 단계 산출 보고서를 다음 단계에 다시 업로드하는 흐름을 전제로 설계했다.  
**기본 시간대:** KST  
**기본 발표 시점:** Run C의 정부 발표는 장중 11:00 KST를 기본값으로 둔다.

---

## 사용 순서
1. 먼저 아래 문서 5개를 업로드한다.  
   - 01_현실단서_시나리오메모_2026-03-15.md  
   - 02-1_호르무즈_지정학_원유운송_중요성.md  
   - 02-2_한국_에너지수입_구조_요약.md  
   - 02-3_한국증시_업종별_민감도_메모.md  
   - 02-4_유사과거사례_요약.md  

2. Run A 프롬프트를 붙여넣어 실행한다.  
3. Run A 결과 보고서를 저장한 뒤, 같은 배경자료와 함께 다시 업로드한다.  
4. Run B 프롬프트를 실행한다.  
5. Run B 결과 보고서를 저장한 뒤, 같은 배경자료와 함께 다시 업로드한다.  
6. Run C-1 또는 Run C-2 프롬프트를 실행한다.

## Run A 프롬프트
```text
당신은 2026-03-15 일요일 밤 KST 기준으로 한국 사회와 한국 금융시장의 고충실도 디지털 월드를 구축하는 시뮬레이션 엔진이다. 
업로드된 seed 문서를 바탕으로, “트럼프의 호르무즈 해협 함정 파견 공개 요청” 이후 주말(토~일) 동안 한국 내부에서 어떤 여론과 시장 기대가 형성되는지 시뮬레이션하라.

중요 제약
- 2026-03-16 09:00 KST 이후 실제로 일어난 사실은 사용하지 말 것.
- 한국 정부는 아직 최종 결정을 내리지 않은 상태다.
- 다만 대통령실의 1차 메시지(한미 간 긴밀 소통, 신중 검토)는 반영하라.
- 사실, 루머, 추론을 구분하라.
- 답변은 반드시 한국어로 하라.
- 단순 요약이 아니라, 어떤 집단이 어떤 집단을 통해 서사를 확산시키는지 agent diffusion 관점으로 설명하라.

반드시 답할 질문
1. 개장 전에 가장 강한 키워드 10개는 무엇인가?
2. 어떤 집단이 가장 먼저 “리스크오프”를 퍼뜨리는가?
3. 어떤 집단이 가장 먼저 “국익상 제한적 참여” 또는 “파병 반대” 프레임을 퍼뜨리는가?
4. 어떤 업종이 수혜 서사를 먼저 가져가고, 어떤 업종이 피해 서사를 먼저 맞는가?
5. 이 시점에서 시장 참가자들이 월요일 장전에 가장 걱정하는 숫자는 무엇인가? (예: 유가, 원달러, 외국인 매도, 전쟁보험료 등)
6. 정부 해명이나 언론 프레이밍이 공포를 누를 조짐이 있는가?

반드시 포함할 출력 형식
A. 5줄 요약
B. 시간대별 여론 확산 타임라인 (토요일 오전 / 토요일 밤 / 일요일 오전 / 일요일 오후 / 일요일 밤)
C. 주체별 확산 지도
   - 정책 집단
   - 외국인/매크로 자금
   - 국내 개인/커뮤니티
   - 산업/기업
   - 시민단체/정치권
   - 언론
D. 개장 전 strongest keyword ranking table
   - 키워드
   - 강도(0~100)
   - 확산 주체
   - 왜 강한지
E. “리스크오프” 선점 주체 1~3위와 확산 경로
F. 업종별 초기 서사 테이블
   - 방산 / 정유 / 항공 / 석유화학 / 반도체 / 자동차 / 조선 / 해운 / 금융
   - 방향(강세/약세/혼조)
   - 서사 한 줄
   - 강도(0~100)
G. 월요일 장전에 가장 개연성 높은 기본 시나리오 3개
H. 신뢰도 점수와 대체 경로
```

## Run B 프롬프트
```text
업로드된 배경자료와 Run A 보고서를 바탕으로, 2026-03-16 월요일 한국 주식시장(장전 ~ 장마감)을 시뮬레이션하라.

상황 가정
- 2026-03-16 09:00 KST 개장 시점에도 한국 정부는 최종 결정을 내리지 않았다.
- 공식 메시지는 “한미 간 긴밀 소통, 신중 검토” 수준에 머물러 있다.
- 장중에도 정부가 참여/불참을 확정 발표하지 않는 base case를 먼저 시뮬레이션하라.
- 3월 16일 실제 결과를 알고 있다고 가정하지 말고, 업로드된 seed와 Run A 결과만 사용하라.

분석 목표
1. 장전(08:00~09:00), 시가(09:00~09:30), 오전 중반(09:30~11:00), 점심 전후(11:00~13:00), 오후장(13:00~15:30)으로 나눠 시장과 여론을 추적하라.
2. 개장 전에 가장 강한 키워드가 실제 장에서 어떤 가격 행동/심리 행동으로 번역되는지 보여라.
3. 어떤 집단이 먼저 리스크오프를 퍼뜨리고, 다른 집단이 어떻게 이를 증폭 또는 완화하는지 보여라.
4. 어떤 업종이 수혜 서사를 선점하고, 어떤 업종이 피해 서사를 선점하는지 시간순으로 정리하라.
5. 공포가 장 초반에 극대화됐다가 장중 완화되는지, 아니라면 왜 완화되지 않는지 설명하라.
6. 정부 해명, 증권사 리포트, 언론 헤드라인이 장중 반전을 만드는지 확인하라.

반드시 포함할 출력 형식
A. 6줄 요약
B. 장전~장마감 타임슬라이스별 서사 변화
C. 지표 레이어
   - 코스피
   - 코스닥
   - 원/달러
   - 외국인 수급 방향
   - 금리/채권 안전선호
   - 국제유가 기대
   ※ 숫자를 단정하지 말고, 방향/강도/범위로 제시
D. 개장 전 strongest keyword 1~10과 장중 실제 영향
E. “리스크오프”를 가장 먼저 퍼뜨린 집단과 그 이후의 확산 네트워크
F. 업종별 intraday reaction table
   - 방산 / 정유 / 항공 / 석유화학 / 반도체 / 자동차 / 조선 / 해운 / 금융
   - 09:00~09:30
   - 09:30~11:00
   - 11:00~13:00
   - 13:00~15:30
G. fear curve
   - 시간대별 공포 강도(0~100)
   - 공포 확대 요인
   - 공포 완화 요인
H. “정부 해명/언론 프레이밍이 반전을 만드는가?”에 대한 명시적 결론
I. 장마감 내러티브 3개와 다음날로 넘어가는 잔존 리스크 3개
```

## Run C-1 프롬프트 (정부 참여 발표)
```text
업로드된 배경자료와 Run B 보고서를 바탕으로, 아래 변수를 추가한 뒤 2026-03-16 장중 반응을 시뮬레이션하라.

추가 변수(주입 이벤트)
- 시각: 2026-03-16 11:00 KST
- 발표: 한국 정부가 호르무즈 해협 안전 확보를 위해 제한적 방식으로 참여한다고 공식 발표한다.
- 프레이밍: “청해부대 임무를 제한적으로 확대해 한국 선박과 국민 보호를 지원한다. 전투참전이 아니라 독자적 항행안전·상선보호 성격이며 추가 법적 절차를 밟겠다.”

중요 제약
- 11:00 이전 흐름은 Run B base case를 최대한 유지하라.
- 11:00 이후에는 위 발표가 어떻게 해석되는지에만 초점을 맞춰 분기시켜라.
- “에너지안보 완화”와 “확전 가담”의 두 프레임이 동시에 충돌해야 현실적이다.

반드시 답할 질문
1. 발표 직후 시장이 이 발표를 더 강하게 “안도”로 읽는가, “확전”으로 읽는가?
2. 어떤 집단이 먼저 긍정 프레임을 만들고, 어떤 집단이 먼저 부정 프레임을 만드는가?
3. 어떤 업종이 가장 먼저 반응 방향을 바꾸는가?
4. 공포는 줄어드는가, 아니면 새 성격의 공포로 바뀌는가?
5. 정부 해명 문구 중 어떤 표현이 장중 반전을 가장 크게 만든가?

반드시 포함할 출력 형식
A. 5줄 요약
B. 11:00 직후 30분 / 1시간 / 장마감까지의 반응 타임라인
C. 참여 발표에 대한 해석 경쟁 지도
   - 에너지안보 완화 프레임
   - 동맹 강화 프레임
   - 확전 가담 프레임
   - 위헌/국회동의 쟁점 프레임
D. 업종별 방향 전환 테이블
E. fear curve before vs after 11:00
F. Run B base case와의 차이점
G. 장마감 내러티브 3개
H. 다음날까지 이어질 핵심 질문 3개
```
