# 릴리즈 소개문 (한국어)

## 제목
MiroFish Codex OAuth 올인원 공개판: Codex OAuth, 한영 UI, 스타터 시나리오를 한 저장소에

## 짧은 소개
이 릴리즈는 MiroFish 본체, 로컬 Codex OAuth 브리지, 한국어/영어 문서, 한국어/영어 UI 업데이트, 그리고 바로 업로드 가능한 예시 시나리오를 하나의 퍼블릭 저장소로 묶은 공개판이다.

## 핵심 포인트
- **저장소 하나만 받으면 됨**: upstream MiroFish와 별도 bridge 저장소를 각각 clone할 필요가 없다.
- **로컬 Codex OAuth 브리지 포함**: `codex login` 기반으로 MiroFish를 로컬 OpenAI 호환 브리지에 연결할 수 있다.
- **한영 문서 및 UI**: 한국어/영어 사용 가이드, 예시 파일, 인터페이스 번역을 포함한다.
- **스타터 시나리오 포함**: 첫 실행 테스트용 현실적인 시나리오 문서와 프롬프트 예시를 넣었다.
- **퍼블릭 공개 안전성 반영**: 비밀키, 토큰, 런타임 로그, 개인 로컬 산출물은 제외했다.

## 이런 사람에게 적합
- 별도 API 구성을 먼저 하지 않고 MiroFish를 시험해보고 싶은 사용자
- 한국어/영어 프롬프트 흐름을 비교하고 싶은 사용자
- 여론/내러티브 시뮬레이션용 시드 문서를 바로 써보고 싶은 사용자

## 참고
- `ZEP_API_KEY`는 직접 준비해야 한다.
- 이 브리지는 **로컬 사용용**이며 퍼블릭 서버리스 운영용이 아니다.
- 먼저 로컬에서 `codex login`이 정상 동작해야 한다.
