# Examples

This repository now includes **two layers of examples**:

1. **Quick-start examples** in `scenarios/` and `prompts/`
   - short seed documents
   - good for the first 10~20 round test
2. **Detailed scenario packs** in `packs/`
   - multi-file background sets
   - modeled after real-world simulation prep notes
   - better when you want a richer MiroFish world with clearer stakeholder and timeline structure

## Quick-start paths
- `scenarios/ko/`, `scenarios/en/`
- `prompts/ko/`, `prompts/en/`

## Detailed pack paths
- `packs/ko/hormuz-korea-2026-03-15/`
- `packs/en/hormuz-korea-2026-03-15/`
- `packs/ko/kakao-openchat-policy-2026-03-15/`
- `packs/en/kakao-openchat-policy-2026-03-15/`
- `packs/ko/tesla-korea-recall-2026-03-15/`
- `packs/en/tesla-korea-recall-2026-03-15/`

## Recommended usage order
1. Start with one file from `scenarios/...`
2. Paste one matching prompt from `prompts/...`
3. Keep the first run small
4. Move to `packs/...` when you want a more detailed, multi-document simulation
