# Simulation prompts (Run A ~ Run C)

## Run A — Sunday-night narrative formation
```text
Using the uploaded seed documents, simulate Korean public and investor-community narrative formation from 21:00 KST on Sunday, March 15, 2026 through 08:50 KST on Monday, March 16, 2026.
Constraints:
- do not assume knowledge of the real Monday outcome
- separate facts from assumptions
- include strongest keywords, first risk-off actors, initial sector narratives, and possible calming frames
Output:
A. five-line summary
B. timeline by time slice
C. strongest-keyword table
D. initial sector narrative table
E. risk-off diffusion map
```

## Run B — pre-open to close, base case
```text
Using the seed documents and the Run A report, simulate the Korean market from pre-open through the close on Monday, March 16, 2026.
Assumptions:
- there is no further hard deployment-size announcement during the morning session
- at 10:40 KST the government only issues a clarification centered on protecting Korean vessels
Must include:
- pre-open / open / late morning / afternoon slices
- directionality for KOSPI / USDKRW / foreign flow
- intraday sector-reaction table
- fear curve
```

## Run C-1 — formalized participation
```text
Starting from Run B, inject a new event at 11:00 KST: the government formalizes limited participation by expanding the mission of an anti-piracy naval unit.
Answer:
- does the market read this more as relief or escalation?

- which sectors flip direction first?
```

## Run C-2 — no additional warship deployment
