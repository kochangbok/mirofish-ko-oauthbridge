# 사용순서 및 파일맵

## 주의
이 폴더는 **가상의 시나리오 팩**이다. 실존 엔티티를 사용하지만, 리콜 규모와 커뮤니티 확산, 후속 해명 흐름은 시뮬레이션용 가정이다.

## 업로드 추천 순서
1. `01_시나리오메모_2026-03-15.md`
2. `02_리콜배경과_브랜드신뢰리스크.md`
3. `03_오너커뮤니티_규제기관_경쟁사반응.md`
4. `04_유사과거패턴과_해석포인트.md`
5. `05_시뮬레이션_프롬프트_RunA-RunC.md`

## 실행 순서
1. 위 5개 파일 업로드
2. Run A로 발표 직후 12시간 반응 시뮬레이션
3. Run A 보고서를 저장/재업로드
4. Run B로 72시간 브랜드/투자자/소비자 반응 시뮬레이션
5. Run B 보고서를 저장/재업로드
6. Run C-1(강한 사과/보상) 또는 Run C-2(최소 대응) 실행
