# 시뮬레이션 프롬프트 (Run A ~ Run C)

## Run A — 발표 직후 12시간
```text
업로드된 seed 문서를 바탕으로, Tesla Korea 리콜 발표 직후 12시간 동안 오너 커뮤니티, 일반 자동차 커뮤니티, 규제기관, EV 유튜브 채널이 어떤 서사를 만드는지 시뮬레이션하라.
필수 출력:
A. 5줄 요약
B. 집단별 반응 표
C. strongest keyword 표
D. 12시간 타임라인
E. 브랜드 신뢰 훼손 1~5위 요인
```

## Run B — 72시간 브랜드/시장 반응
```text
업로드된 seed 문서와 Run A 보고서를 바탕으로, 72시간 동안 브랜드 신뢰, 소비자 반응, 투자자 반응, 경쟁사 수혜 서사를 시뮬레이션하라.
반드시 포함:
- 오너 vs 비오너 서사 차이
- 국토교통부 발표 효과
- 현대차/기아 반사수혜 가능성
- 브랜드 회복/악화 트리거
```

## Run C-1 — 강한 사과와 보상 발표
```text
Run B 결과를 바탕으로, Tesla Korea가 24시간 내 무상점검 확대, 배터리 관련 추가 보증, 대차지원, CEO 영상 사과를 발표한다고 가정하라.
반드시 답할 것:
- 어떤 집단이 먼저 신뢰를 회복하는가
- 어떤 집단은 여전히 부정적인가
- 어떤 문구가 가장 효과적인가
```

## Run C-2 — 최소 대응 유지
```text
Run B 결과를 바탕으로, Tesla Korea가 '예방적 리콜이며 과도한 우려는 불필요하다'는 최소한의 메시지만 반복한다고 가정하라.
반드시 답할 것:
- 오너/비오너 갈등이 어떻게 심화되는가
- 규제기관 개입 서사가 커지는가
- 브랜드 손상이 얼마나 길게 남는가
```
