# 시뮬레이션 프롬프트 (Run A ~ Run C)

## Run A — 유출본 확산
```text
업로드된 seed 문서를 바탕으로, 2026-03-15 18:00 KST 유출 시점부터 23:00 KST 공식 발표 직전까지 어떤 커뮤니티가 어떤 문장을 중심으로 반발을 키우는지 시뮬레이션하라.
필수 출력:
A. 5줄 요약
B. 커뮤니티별 반응 표
C. strongest keyword 표
D. 유출본 확산 타임라인
E. 카카오가 가장 위험하게 보이는 문장 5개
```

## Run B — 공식 공지 후 24시간
```text
업로드된 seed 문서와 Run A 보고서를 바탕으로, 공식 공지 직후부터 24시간 동안 카카오 오픈채팅 정책 반응을 시뮬레이션하라.
반드시 포함:
- DC Inside / Blind / Naver Cafe / X / 기자 / 유튜버 반응 비교
- 이탈 의도와 대체 플랫폼 언급
- 규제기관/정치권 프레임 개입 여부
- PR 문구의 진정 효과 vs 악화 효과
```

## Run C-1 — 사과 및 완화 발표
```text
Run B 결과를 바탕으로, 카카오가 다음날 11:00 KST에 “보관기간 단축 검토, 운영자 예외 절차, 데이터 열람 제한”을 포함한 완화 조치를 발표한다고 가정하라.
반드시 답할 것:
- 어떤 집단이 가장 먼저 안도하는가
- 여전히 떠나려는 집단은 누구인가
- 어떤 문구가 가장 큰 진정 효과를 내는가
```

## Run C-2 — 강행 및 강경 해명
```text
Run B 결과를 바탕으로, 카카오가 다음날 11:00 KST에 “이 조치는 예정대로 시행되며 이용자 안전을 위해 필수적”이라고 강경 해명한다고 가정하라.
반드시 답할 것:
- 반발이 어떤 정치적/사회적 프레임으로 확장되는가
- 대체 플랫폼 이동 담론이 실제 행동 서사로 바뀌는가
- 브랜드 신뢰가 얼마나 오래 손상되는가
```
