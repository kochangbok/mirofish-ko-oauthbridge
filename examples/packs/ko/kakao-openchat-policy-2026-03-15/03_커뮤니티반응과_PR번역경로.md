# 커뮤니티 반응과 PR 번역 경로

## 전파 경로
1. 내부 초안 캡처 유출
2. DC Inside / Blind / X에서 자극적 문장만 추출해 재배포
3. 기자와 IT 유튜버가 “논란 요약” 콘텐츠 생성
4. 카카오 공식 공지가 올라오지만, 이미 유출본 해석이 기준 프레임이 됨
5. 후속 해명문이 나오면 ‘사과’로 읽히는지 ‘변명’으로 읽히는지에 따라 반응 분기

## 집단별 기본 가설
- 오픈채팅 운영자: 가장 강한 반발 가능성
- 직장인 익명방 사용자: 익명성 붕괴 프레임에 민감
- 일반 이용자: 프라이버시보다는 “불편/귀찮음”과 “왠지 찜찜함”으로 반응할 수 있음
- IT 기자/유튜버: 논쟁의 핵심 문장을 압축해 확산
- 규제기관/정치권: 청소년 보호·범죄 예방 vs 과잉 규제 논쟁으로 가져갈 수 있음

## MiroFish 출력 요구
- 커뮤니티별 반발 강도(0~100)
- 이탈 의도 / 대체 플랫폼 언급 강도
- 카카오 문구 중 악화시키는 표현 / 완화시키는 표현
- 48시간 뒤 잔존 리스크
