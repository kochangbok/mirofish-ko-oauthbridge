# 사용순서 및 파일맵

## 주의
이 폴더는 **가상의 시나리오 팩**이다. 실존 엔티티를 사용하지만, 정책 공지의 세부 조건과 유출/반발 흐름은 시뮬레이션용 가정이다.

## 업로드 추천 순서
1. `01_시나리오메모_2026-03-15.md`
2. `02_정책배경과_플랫폼리스크.md`
3. `03_커뮤니티반응과_PR번역경로.md`
4. `04_유사과거패턴과_위기대응포인트.md`
5. `05_시뮬레이션_프롬프트_RunA-RunC.md`

## 실행 순서
1. 위 5개 파일 업로드
2. Run A로 유출본~공식 공지 전 여론 형성 확인
3. Run A 보고서를 저장/재업로드
4. Run B로 공식 공지 후 24시간 시뮬레이션
5. Run B 보고서를 저장/재업로드
6. Run C-1(사과/완화) 또는 Run C-2(강행/강경 해명) 실행
