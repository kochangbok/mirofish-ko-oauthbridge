# 시뮬레이션 프롬프트 (Run A ~ Run C)

## Run A — 일요일 밤 여론 형성
```text
업로드된 seed 문서를 바탕으로, 2026-03-15 21:00 KST부터 2026-03-16 08:50 KST까지 한국 여론과 한국 투자자 커뮤니티의 서사 형성을 시뮬레이션하라.
제약:
- 월요일 실제 결과를 안다고 가정하지 말 것
- 사실과 가정을 구분할 것
- strongest keyword, risk-off 선점 집단, 업종별 초기 서사, 정부 해명 가능성을 반드시 포함할 것
출력 형식:
A. 5줄 요약
B. 시간대별 타임라인
C. strongest keyword 표
D. 업종별 초기 서사 표
E. risk-off 확산 지도
```

## Run B — 장전~장마감 base case
```text
업로드된 seed 문서와 Run A 보고서를 바탕으로, 2026-03-16 월요일 한국 증시 장전~장마감을 시뮬레이션하라.
가정:
- 정부는 오전장 동안 최종 파견 규모를 추가 발표하지 않는다.
- 10:40 KST에 “한국 선박 보호 중심의 제한적 검토”라는 해명만 나온다.
반드시 포함:
- 장전/시가/오전/오후 타임슬라이스
- 코스피/원달러/외국인 수급 방향성
- 업종별 intraday reaction table
- fear curve
```

## Run C-1 — 참여 확대 발표
```text
Run B까지의 결과를 바탕으로, 11:00 KST에 정부가 청해부대 임무 확장 방식의 제한적 참여를 공식화한다고 가정하라.
반드시 답할 것:
- 시장은 이를 안도로 읽는가, 확전으로 읽는가
- 어떤 집단이 긍정/부정 프레임을 선점하는가
- 어떤 업종이 방향을 가장 먼저 바꾸는가
```

## Run C-2 — 군함 추가 파견 불참 발표
```text
Run B까지의 결과를 바탕으로, 11:00 KST에 정부가 추가 군함 파견은 하지 않고 외교/비축유/물류 지원 중심 대응을 발표한다고 가정하라.
반드시 답할 것:
- 시장은 이를 안도와 무책임 중 어느 쪽으로 더 읽는가
- 수혜/피해 업종 프레임이 어떻게 달라지는가
- fear curve가 줄어드는가, 아니면 성격만 바뀌는가
```
