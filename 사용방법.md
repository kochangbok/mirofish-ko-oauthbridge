# 사용방법

이 저장소에는 **MiroFish를 Codex OAuth(로컬 ChatGPT/Codex 로그인 세션)** 로 연결하기 위한 로컬 브리지 `codex-bridge`가 들어 있습니다.

## 빠른 시작

1. `codex` CLI 설치
2. `codex login` 완료
3. `codex-bridge` 실행
4. MiroFish `.env`에서 `LLM_BASE_URL`을 로컬 브리지로 지정
5. MiroFish backend/frontend 실행

## 문서 위치

- 브리지 개요: [`codex-bridge/README.md`](./codex-bridge/README.md)
- 상세 사용법: [`codex-bridge/사용방법.md`](./codex-bridge/사용방법.md)

## 핵심 설정 예시

```env
LLM_API_KEY=local-codex-bridge
LLM_BASE_URL=http://127.0.0.1:8787/v1
LLM_MODEL_NAME=gpt-5.1-codex-mini
ZEP_API_KEY=your-zep-api-key
```

## 주의

이 브리지는 **로컬 Codex 로그인 세션**과 `codex exec`에 의존하므로,
**Vercel 같은 퍼블릭 서버리스 환경에 그대로 배포하는 용도는 적합하지 않습니다.**

권장 사용처는 다음과 같습니다.
- 개인 실험
- 내부용 프로토타입
- 로컬 개발 환경 연동
