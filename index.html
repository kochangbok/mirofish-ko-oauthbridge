로컬 전용 브리지

codex-bridge

MiroFish를 Codex OAuth(로컬 ChatGPT/Codex 로그인 세션)로 연결하기 위한 OpenAI 호환 브리지입니다.

핵심 요약

  • POST /v1/chat/completions 형태의 요청을 받습니다.
  • 내부적으로 codex exec를 호출합니다.
  • 로컬 로그인 세션에 의존하므로 개인/내부 테스트용에 적합합니다.
  • Vercel에 배포된 이 페이지는 문서 안내용이며, 브리지 자체는 로컬에서 실행해야 합니다.

빠른 시작

cd codex-bridge
npm install

PORT=8787 \
CODEX_MODEL=gpt-5.1-codex-mini \
CODEX_BRIDGE_WORKDIR=/path/to/MiroFish \
npm start

MiroFish .env 예시:

LLM_API_KEY=local-codex-bridge
LLM_BASE_URL=http://127.0.0.1:8787/v1
LLM_MODEL_NAME=gpt-5.1-codex-mini
ZEP_API_KEY=your-zep-api-key

주의

이 프로젝트는 로컬 Codex 로그인 세션을 사용합니다. 따라서 퍼블릭 서버리스 환경에서 브리지 자체를 그대로 운영하는 방식은 보안/구조상 권장되지 않습니다.
